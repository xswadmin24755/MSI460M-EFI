# MSI460M-EFI
这是黑苹果的EFI,基于opencore0.7.1
## 使用的配置
MSI 460M BAZOOKA. 

CPU:i7 10700K. 

RAM:16GB. 

SSD:1024GB. 

Display:Intel UHD 630. 

所有硬件均可正常驱动,核显加速,显存正常,[跑分图](https://azure.ytsu.xyz).
